# 📊 Анализ рынка видеоигр для планирования маркетинговой кампании

## 📌 Описание проекта

Проведен анализ рынка видеоигр с целью определения факторов успеха игр и формирования рекомендаций для маркетинговой кампании.

---

## 🎯 Бизнес-задача

Определить:

- какие игры наиболее успешны  
- какие факторы влияют на продажи  
- на какие сегменты стоит ориентироваться при запуске кампании  

---

## 📊 Данные

Использован датасет с информацией о видеоиграх:

- продажи по регионам  
- жанры  
- платформы  
- оценки пользователей и критиков  
- год выпуска  

---

## 🔬 Подход

- Очистка данных  
- Анализ распределений  
- Анализ продаж по:
  - платформам  
  - жанрам  
  - регионам  
- Исследование влияния оценок на продажи  
- Выявление трендов  

---

## 📈 Результаты

- Популярность платформ и жанров меняется со временем  
- Некоторые жанры стабильно показывают высокие продажи  
- Оценки критиков умеренно коррелируют с продажами  
- Региональные предпочтения различаются  

---

## ⚠️ Интерпретация

- Успех игры зависит от комбинации факторов  
- Нельзя ориентироваться только на один показатель  
- Важно учитывать региональные особенности  

---

## 💡 Вывод и рекомендации

👉 Для маркетинговой кампании:

- фокусироваться на популярных жанрах  
- учитывать региональные предпочтения  
- выбирать актуальные платформы  
- не переоценивать влияние пользовательских оценок  

---

## 📌 Практическое применение

- планирование маркетинговых кампаний  
- выбор целевой аудитории  
- оптимизация продуктовой стратегии
