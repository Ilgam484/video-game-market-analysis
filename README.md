# 📊 Анализ рынка видеоигр для планирования маркетинговой кампании

## 📌 Контекст

Вы — аналитик в онлайн-магазине видеоигр.  
Ваша задача — проанализировать рынок и определить стратегию маркетинговой кампании для максимизации продаж.

---

## 🎯 Бизнес-задача

Определить:

- какие игры наиболее успешны  
- какие факторы влияют на продажи  
- на какие сегменты стоит ориентироваться при запуске маркетинговой кампании  

---

## 📊 Данные

Использован датасет с информацией о видеоиграх:

- продажи по регионам (NA, EU, JP, Other)  
- жанры игр  
- платформы  
- оценки пользователей и критиков  
- год выпуска  

---

## 🔬 Подход

- Очистка и предобработка данных  
- Анализ распределений  
- Анализ продаж по:
  - платформам  
  - жанрам  
  - регионам  
- Исследование влияния оценок на продажи  
- Выявление трендов во времени  

---

## 📈 Результаты

- Популярность платформ меняется со временем  
- Некоторые жанры стабильно показывают высокие продажи  
- Продажи существенно различаются по регионам  
- Оценки критиков умеренно коррелируют с продажами  
- Пользовательские оценки не являются надежным предиктором успеха  

---

## ⚠️ Интерпретация

- Успех игры определяется комбинацией факторов  
- Нельзя ориентироваться только на один показатель (например, рейтинг)  
- Региональные различия существенно влияют на продажи  

---

## 💡 Вывод и рекомендации

👉 Для маркетинговой кампании:

- Фокусироваться на популярных жанрах (Action, Shooter)  
- Основные регионы: Северная Америка и Европа  
- Приоритетные платформы: PlayStation и Xbox  
- Не ориентироваться только на пользовательские оценки  

👉 Рекомендуется строить маркетинговую стратегию с учетом этих факторов для максимизации продаж

---

## 📌 Ключевой вывод

Продажи игр определяются комбинацией жанра, платформы и региона.  
Эффективная маркетинговая стратегия должна учитывать все три фактора.

---

## 📌 Практическое применение

- планирование маркетинговых кампаний  
- выбор целевой аудитории  
- оптимизация стратегии продаж
