# Анализ рынка видеоигр для планирования кампании (Стримчик)

## Цель проекта
На основе исторических данных о продажах игр, оценках пользователей и критиков, жанрах и платформах выявить закономерности успешности игры.  
Результаты используются для выбора перспективных продуктов и планирования рекламной кампании на следующий год (условно: декабрь 2016 → план на 2017).

## Данные
Источник: открытые данные о видеоиграх и продажах (до 2016 года).

Основные поля:
- `name` — название игры
- `platform` — платформа (Xbox, PlayStation и др.)
- `year_of_release` — год выпуска
- `genre` — жанр
- `na_sales`, `eu_sales`, `jp_sales`, `other_sales` — продажи по регионам (млн копий)
- `critic_score` — оценка критиков (0–100)
- `user_score` — оценка пользователей (0–10, встречается значение `tbd`)
- `rating` — возрастной рейтинг ESRB

> Примечание: данные за 2016 год могут быть неполными.

## Инструменты
- Python: pandas, numpy, scipy
- Визуализация: matplotlib, seaborn
- Jupyter Notebook

## Что было сделано
- Подготовка данных:
  - приведение названий столбцов к нижнему регистру
  - преобразование типов данных (годы, рейтинги)
  - обработка пропусков и значения `tbd` в `user_score`
  - расчёт глобальных продаж (сумма продаж по регионам)
- Исследовательский анализ (EDA):
  - динамика выпуска игр по годам
  - анализ жизненного цикла платформ и изменений продаж по платформам
  - выбор актуального периода для прогноза/планирования
  - сравнение продаж по платформам (в т.ч. boxplot по глобальным продажам)
  - влияние оценок критиков и пользователей на продажи (scatter + корреляция)
  - анализ жанров и их прибыльности
- Региональный анализ:
  - портрет пользователя для регионов NA / EU / JP:
    - топ платформ
    - топ жанров
    - влияние рейтинга ESRB на продажи
- Проверка гипотез:
  - сравнение средних пользовательских рейтингов платформ Xbox One и PC
  - сравнение средних пользовательских рейтингов жанров Action и Sports
  - выбор уровня значимости `alpha`, формулировка H0/H1 и применение статистического критерия

## Ключевые выводы
- Определён актуальный период данных для планирования кампании
- Выделены перспективные платформы, растущие или сохраняющие спрос
- Показано, как оценки критиков/пользователей связаны с продажами (в разрезе платформ)
- Определены жанры с наиболее высокими и низкими продажами
- Составлены портреты региональных пользователей и выявлены различия по платформам/жанрам
- Проверены продуктовые гипотезы с использованием статистических тестов
